Sobre o Projeto

Objetivo

Analisar dados de batalhas do Clash Royale em MongoDB para balanceamento do jogo.

Dados

Jogadores (nickname, clãs, troféus) e batalhas (cartas, decks, vencedor, torres).

Fonte

API Oficial do Clash Royale.
